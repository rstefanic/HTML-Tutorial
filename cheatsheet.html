<p> tags

<p> tags allow you to insert a paragraph into your page.

Typically, if you have a block of text, then you'll want to use a <p> tag.

Closing a <p> tag will automatically create a line break after it.

Here is some example code: <p>This is a paragraph tag.</p>

<a> tags

<a> tags allow you to create links that can be placed inside of your pages.

Closing an <a> tag will not cause a line break.

An <a> tag must have an href="" inside of the opening <a> tag. It's the reference point. The address listed in the <a> tag will be where the user is redirected once they've clicked on the link

Here is an example of an <a> tag. This links to Google's website.:

<a href="https://www.google.com">www.google.com</a>

<h1> tags

<h1> tags allow you to create headings.

<h1> tags are the largest heading tags.

An <h1> isn't the only heading. You can use anything from <h1> to <h5>, although h1-h3 are the most common ones.

Here is an example of an <h1> tag:

<h1>Heading</h1>

Closing Remarks on Text Properties

There are a few extra tags that are related to text.

The <strong> tag, the <em> tag, and the <span> tag.

<strong> tags are used to bold text. Example: <strong>Bolded text</strong>

<em> tags are for emphasis (italics). Example: <em>Emphasised text</em>

<span> tags are a little special, and they will be covered in more deail later
